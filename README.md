# CSV-Date-Validation
This project is a simple Spring Boot web application that accepts a CSV file upload, validates each row, and returns a structured JSON response with success or error details.
run-application step-by-step
download csv-validator (2).zip this file.
1 Step:- Extract all this file (csv-validator (2).zip).
2 Step:- Open any editor like Intellij IDLE or Springboot editor.
3 Step:- The next step is to run this file (CsvValidatorApplication)the application is started.
4 Step:- Go to any browser and run this URL( http://localhost:8080/index.html ).
5 Step:- Save the provided CSV file (sample-data.csv). 
6 Step:- Click the CHOOSE FILE button and upload a file.
7 Step:- Next step click the UPLOAD AND VALIDATE button.
